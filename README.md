# 📊 Data Visualization & Analytics Dashboard

## Overview
This project is a simple interactive dashboard built with **Streamlit** and **Plotly**.  
It allows users to upload datasets, visualize distributions, scatter plots, and view summary statistics.

## Features
- Upload CSV files
- Interactive charts (histogram, scatter plot)
- Summary statistics
- Data cleaning utilities

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## How to Run
```bash
pip install streamlit pandas plotly
streamlit run dashboard_app.py
